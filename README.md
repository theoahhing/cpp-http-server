# cpp-http-server

A lightweight HTTP server built from scratch in C++ using raw sockets.

## Goals
- Learn low-level networking (TCP sockets)
- Implement HTTP request parsing manually
- Build a clean backend architecture in C++
- Demonstrate systems-level programming skills

## Planned Features
- TCP socket server
- HTTP request parsing
- Routing system
- JSON responses
- Multi-client handling
- Logging
